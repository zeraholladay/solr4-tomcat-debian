# solr4-tomcat

Installs Solr 4.1.0 in Tomcat 6.  See http://lucene.apache.org/solr/.

## Installation

Method (1) install tomcat6:

    apt-get install tomcat6
    dpkg -i solr4-tomcat.deb

Or method (2) try this:

    dpkg -i mypreferredsoftware.deb
    apt-get install -f
